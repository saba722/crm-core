# 🧑‍💻 FullStack CRM

A modern full-stack CRM (Customer Relationship Management) application built with **ASP.NET Core**, **React**, and modern development tools.

This project is currently in the setup phase and will gradually evolve into a complete CRM platform with authentication, customer management, dashboards, analytics, and more.

---

# 🚀 Tech Stack

## Backend

* ASP.NET Core (.NET)
* C#
* REST API

## Frontend (planned)

* React
* JavaScript / TypeScript
* Tailwind CSS

## Tools & Dev Environment

* Visual Studio Code
* Git & GitHub
* Node.js

# ⚙️ Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/saba722/fullstack-crm.git
```

## 2. Open Project

```bash
cd fullstack-crm
code .
```

---

# 🛠 Backend Setup (.NET)

## Install .NET SDK

Download from:
[https://dotnet.microsoft.com/download](https://dotnet.microsoft.com/download)

Verify installation:

```bash
dotnet --version
```

## Run Backend

```bash
dotnet run
```

---

# 🌐 Frontend Setup (React)

## Install Node.js

Download from:
[https://nodejs.org/](https://nodejs.org/)

Verify installation:

```bash
node -v
npm -v
```

## Install Dependencies

```bash
npm install
```

## Start Frontend

```bash
npm start
```
